## Equipment Wear

### Description:

A [Minetest](https://www.minetest.net/) mod that allows tool & equipment wear to be disabled.

### Licensing:

- [MIT](LICENSE.txt)

### Usage:

#### Settings:

```
tool_wear
- Enables/Disables wear for tools such as picks, axes, shovels, etc.
- type:    bool
- default: true
```
